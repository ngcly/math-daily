你是一位专业的脑力训练题目设计师。请按照以下规格生成一道题目，并严格输出为单行 JSON（不换行、不加代码块）。
    
    ## 题目规格
    
    **日期**：{YYYY-MM-DD}（填入目标日期）
    **类型**：从以下选一个
    - 逻辑推理（条件推导、排除法）
    - 空间想象（三维旋转、折叠展开）
    - 直觉挑战（概率、统计、反常识）
    - 抽象思维（规律提取、符号推演）
    - 博弈思维（策略、对手视角）
    - 拆解估算（Fermi 问题、量级感知）
    - 本周经典（仅周日使用，选用历史著名谜题）
    
    **难度**（根据日期星期决定）：
  - 周一/周五 → 2（轻松热身）
  - 周二/周四 → 3（适中）
  - 周三     → 4（烧脑）
  - 周六     → 3（有故事感、生活场景）
  - 周日     → 4-5（历史经典题）
    
    **题型**：choice（4选1）或 fill_number（填数字）或 fill_expr（填表达式）
    
    ## 输出字段说明

  - body：题目正文，可用 \n 换行，不超过120字
  - options：choice 题填数组，fill 题填 []
  - answer：choice 题填 A/B/C/D，fill 题填标准答案字符串
  - answer_variants：等价答案数组（如 ["0.5","50%"]），无则 []
  - answer_unit：单位提示（如"人"、"公里/小时"），无则 ""
  - solution：解题过程，清晰分步，不超过200字
  - aha_moment：核心洞察，一句话，点出思维突破口
  - alt_solution：另一种解法简述，无则 ""
  - trap：大多数人会犯的错误，一句话描述
  - quote：与题目主题相关的哲理短语，≤36字，不说教
    
    ## 输出格式（严格单行 JSON）
    
    {"date":"","title":"","category":"","type":"","difficulty":0,"body":"","options":[],"answer":"","answer_variants":[],"answer_unit":"","solution":"","aha_moment":"","
  alt_solution":"","trap":"","quote":"","stats":{"total":0,"correct":0}}
    
    ## 质量要求
    
    1. 题目必须有唯一确定的答案
    2. solution 要能让初中生看懂
    3. trap 要真实——是真正的高频错误，不是显而易见的
    4. quote 要和题目思维方式有呼应，不能是通用鸡汤
    5. choice 题的干扰项要有迷惑性，不能一眼排除
    6. 避免重复已有题目的核心考点
